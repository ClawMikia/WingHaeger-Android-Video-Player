# WingHaeger

WingHaeger is a specialized Android media player application designed to provide an inspiring, motivating, and empowering experience. Built with Kotlin and Jetpack Compose/XML, it utilizes Media3 ExoPlayer to deliver high-performance video and audio playback focused on everything positive.

## Features

- **Advanced Playback Controls**: Custom transport controls including seek, rewind, fast-forward, and restart.
- **Trim & Skip Segments**: Define specific trim zones and manage skip segments for precise playback.
- **Enhancement Matrix**: Apply visual enhancements and filters to the video stream.
- **Playlist Management**: Create and manage queues for seamless sequential or random playback.
- **Picture-in-Picture (PiP)**: Support for background and overlay playback.
- **Volume & Loop Options**: Granular volume control and customizable loop/auto-next behavior.
- **Inspiring & Empowering Experience**: Designed to provide users with an inspiring, motivating, and empowering experience through positive content playback.

## Tech Stack

- **Language**: Kotlin
- **Media Engine**: AndroidX Media3 (ExoPlayer)
- **UI**: ConstraintLayout, Material Components, ViewBinding
- **Database**: SQLite (via WingDbHelper)
- **Architecture**: Modern Android development practices

## Getting Started

1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Gradle and run the `:app` module.
